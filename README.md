# Hebat East - v1.0 beta

## ⚠️⚠️ Version Notice

**This is the first experimental prototype of the Hibat East website.**  
The current production version is available at: `https://hebateast.com/` also : https://hebat-east-code.vercel.app/

---

## Tech Stack

- **React 18** + **TypeScript 5**
- **Vite 5** (build tool)
- **Tailwind CSS 3** (styling)
- **React Router** (navigation)
- **shadcn/ui** (UI components)

---

## Core Features

- Bilingual support (Arabic RTL / English LTR)
- Responsive design
- WhatsApp contact integration
- Lazy-loaded images

---

## Project Structure
```
hibat-east/
├── src/
│   ├── components/       # Reusable UI components
│   ├── pages/           # Application pages (Home, About, Services, etc.)
│   ├── context/         # Language management
│   └── App.tsx          # Main router
├── public/              # Static assets
└── vite.config.ts
```
